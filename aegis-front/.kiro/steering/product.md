# Product Overview

AEGIS RPA Frontend is a Flutter-based desktop application that provides an intuitive interface for commanding and monitoring the AEGIS RPA Backend's cognitive automation engine.

## Core Purpose

Enable users to submit natural language task instructions and monitor real-time execution progress through a clean, three-screen flow with WebSocket-based live updates.

## Key Features

- Simple task input using natural language
- Real-time progress monitoring with WebSocket updates
- Execution history with detailed session results
- Material 3 design with dark mode support
- Reactive state management using Provider/Riverpod
- Robust error handling with automatic retry logic

## User Flow

1. **Onboarding** (first-time users): Introduction to AEGIS capabilities
2. **Landing Screen**: Submit automation tasks via text input
3. **Task Execution Screen**: Monitor live progress with subtask cards
4. **History View**: Review past automation sessions

## Target Platform

Desktop applications (Windows, macOS, Linux) with potential for mobile expansion.
